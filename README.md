# AI Swing Trading Signal Generator

This project provides a system to generate trading signals for swing trading on Binance Futures.

## Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/ai-swing-trading.git
    cd ai-swing-trading
    ```

2.  **Create a virtual environment:**
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows, use `venv\Scripts\activate`
    ```

3.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

## Configuration

1.  **Create a `.env` file** from the example:
    ```bash
    cp .env_example .env
    ```

2.  **Edit the `.env` file** and add your Binance API keys if you want to use live data. For testing, the default settings are fine.

## Important: Training vs. Inference Environments

This project is designed to run in two distinct modes, which you can set in `config/settings.yaml`:

-   **`environment: "inference"` (Default)**: A lightweight mode for generating signals on a local machine. It disables heavy MLOps tools like MLflow to save resources. Use this mode for running the `auto_signal.py` bot.
-   **`environment: "training"`**: A heavyweight mode for training the AI model, typically on a powerful cloud server or Colab. This mode enables MLflow for experiment tracking. Use this for `auto_train.py`.

For more details, please read the [Environment Guide](docs/environments.md).

## How to Run

To generate a trading signal, run the test script:

```bash
python scripts/test_signal.py
```

This will fetch the latest 1-hour data for BTCUSDT, calculate all the indicators, and generate a trading signal.

## How to Use Signals for Manual Trading on Binance

1.  **Receive the Signal:** The script will output a signal like "BUY", "SELL", or "NEUTRAL", along with entry price, take profit (TP) levels, and a stop loss (SL).

2.  **Open Binance Futures:** Log in to your Binance account and navigate to the Futures trading section.

3.  **Select the Pair:** Make sure you have selected the correct trading pair (e.g., BTCUSDT).

4.  **Place the Order:**
    *   **Entry:** Place a Limit order at the suggested `Entry Price`.
    *   **Take Profit:** You can set multiple TP orders at the `TP1`, `TP2`, and `TP3` levels.
    *   **Stop Loss:** Set a Stop Market or Stop Limit order at the suggested `SL` price to protect your capital.

5.  **Manage the Trade:** Once your entry order is filled, monitor the trade. You can close the position when it hits one of the TP levels or if you want to exit manually.

**Disclaimer:** Trading cryptocurrency involves significant risk. The signals generated by this system are for informational purposes only and should not be considered financial advice. Always do your own research and manage your risk appropriately.
